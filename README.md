Ekart House Rent is a web application built using HTML, CSS, and EJS for the frontend and Node.js with Express for the backend.
It allows users to view, add, and manage house rental listings with features like creating new listings, viewing details, and managing data dynamically through EJS templates.

Tech Stack:

#Frontend: HTML, CSS, EJS
Backend: Node.js, Express.js
Database: MongoDB

Features:
CRUD OPERATIONS
Add new rental listings
View available houses
Responsive layout and clean UI
